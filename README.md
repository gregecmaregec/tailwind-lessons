## Tailwind CSS Mastery

A hands-on learning course for Tailwind CSS, built with Next.js, which will already come pre-installed with Tailwind. 


## Structure

7 modules, 33 lessons total.

**Module 1 - Fundamentals**
- Spacing & Padding
- Width, Height & Sizing
- Colors & Backgrounds
- Typography & Text Styling
- Borders & Rounded Corners

**Module 2 - Layout**
- Flexbox Fundamentals
- Advanced Flexbox Patterns
- CSS Grid Basics
- Advanced Grid Layouts
- Positioning & Z-Index

**Module 3 - Responsive Design**
- Mobile-First Approach
- Breakpoints & Media Queries
- Responsive Typography
- Container Queries

**Module 4 - Colors & Effects**
- Color System Deep Dive
- Gradients & Patterns
- Shadows & Elevation
- Opacity & Blend Modes
- Filters & Backdrop Effects

**Module 5 - Advanced Techniques**
- Transitions & Timing
- Transforms & 3D Effects
- Animations & Keyframes
- Dark Mode
- Custom Utilities & Plugins

**Module 6 - Components**
- Buttons & Links
- Cards & Containers
- Forms & Inputs
- Navigation & Menus
- Modals & Overlays
- Landing Page

**Module 7 - Best Practices**
- Component Organization
- Performance Optimization
- Accessibility
